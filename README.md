# 🚀 PathPilot — Predict Your Future Before You Waste It

> **Your browser shows what you’re doing. PathPilot shows where it leads.**


## ⚡ What is PathPilot?

PathPilot is a **Hermes-powered AI Chrome extension** that doesn't just track behavior — it **simulates your future in real time**.

Every action you take in your browser is evaluated against your life goal.

Then PathPilot shows you:

* 📉 Where your current habits lead in 18 months
* 🔥 When you burn out
* 💸 When you run out of money
* 🧠 What investors / mentors would say about you

And it delivers it through **cinematic voice narration**.

👉 This is not productivity tracking.
👉 This is **decision enforcement through future simulation**.

---

## 💥 Core Idea

Most tools tell you what you did.

**PathPilot shows you what it will cost you.**

Instead of:

> “You spent 2h on YouTube”

You get:

> “If you continue this pattern → you fail in 11 months.”

That changes behavior instantly.

## 🛡️ Safety & Scam Protection

PathPilot actively protects users from harmful or low-value browsing.

* 🚫 Detects scam / phishing / fake crypto sites
* ⚠️ Flags high-risk pages in real-time
* 🧠 AI evaluates intent (waste vs productive vs dangerous)
* 🔒 Optional blocking for unsafe or distracting sites

Instead of just warning, PathPilot explains:

> “This site is likely a scam or time-waste. Continuing increases risk of loss or distraction.”

👉 This turns passive browsing into **active decision awareness**

## 🧠 AI Architecture

```
Browser Event
   ↓
L0 — Rule Engine (FREE, instant decisions)
   ↓
L1 — Kimi K2 (real-time page understanding)
   ↓
L2 — Kimi K2.5 (future simulation engine)
   ↓
L3 — Hermes 3 70B (agent decisions)
   ↓
L4 — ElevenLabs (voice narration)
```

---

## 🤖 Model Stack

| Layer | Model        | Purpose                         |
| ----- | ------------ | ------------------------------- |
| L0    | Rule System  | Instant classification (0 cost) |
| L1    | Kimi K2      | Page understanding              |
| L2    | Kimi K2.5    | Future simulation               |
| L3    | Hermes 3 70B | Autonomous agent logic          |
| L4    | ElevenLabs   | Voice narration                 |

---

## 💸 Cost Optimization

* 80–90% handled by **rule engine (FREE)**
* AI only used when necessary
* Heavy models are **user-triggered only**

👉 Built for real-world daily use, not demo-only

---

🛠 Tech Stack
Chrome Extension (Manifest V3)
Node.js + Express
OpenRouter (multi-model routing)
Kimi K2 / K2.5
Hermes 3 (Nous Research)
🔊 Voice System
ElevenLabs (primary cinematic voice)
Browser TTS (fallback if API fails)

👉 Ensures voice feedback always works, even without external API
